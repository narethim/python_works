# Chapter 10 - Files and Exceptions

## Preparation

```sh
chapter_10$ python -m venv chapter11-env
source chapter11-env/bin/activate
(chapter11-env) chapter_10$ pip list

(chapter11-env) chapter_10$ pip install --upgrade pip

(chapter11-env) chapter_10$ pip list

(chapter11-env) chapter_10$ pip install pytest

(chapter11-env) chapter_10$ pip list

(chapter11-env) chapter_10$ pip freeze > requirements.txt
```

Install packages from the `requirements.txt`

```sh
chapter_10$ python -m venv chapter11-env2
chapter_10$ source chapter11-env2/bin/activate

(chapter11-env2) chapter_10$ pip install --upgrade pip
(chapter11-env2) chapter_10$ pip install -r requirements.txt
(chapter11-env2) chapter_10$ pip list
```
