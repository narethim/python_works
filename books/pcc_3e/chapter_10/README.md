# Chapter 10 - Files and Exceptions

## Preparation

```sh
chapter_10$ python -m venv chapter10-env
source chapter10-env/bin/activate
(chapter10-env) chapter_10$ pip list

(chapter10-env) chapter_10$ pip install --upgrade pip

(chapter10-env) chapter_10$ pip list

(chapter10-env) chapter_10$ pip install pytest

(chapter10-env) chapter_10$ pip list

(chapter10-env) chapter_10$ pip freeze > requirements.txt
```

Install packages from the `requirements.txt`

```sh
chapter_10$ python -m venv chapter10-env2
chapter_10$ source chapter10-env2/bin/activate

(chapter10-env2) chapter_10$ pip install --upgrade pip
(chapter10-env2) chapter_10$ pip install -r requirements.txt
(chapter10-env2) chapter_10$ pip list
```
