## Preparation

```sh
chapter_09$ python -m venv chapter11-env
source chapter11-env/bin/activate
(chapter11-env) chapter_09$ pip list

(chapter11-env) chapter_09$ pip install --upgrade pip

(chapter11-env) chapter_09$ pip list

(chapter11-env) chapter_09$ pip install pytest

(chapter11-env) chapter_09$ pip list

(chapter11-env) chapter_09$ pip freeze > requirements.txt
```

Install packages from the `requirements.txt`

```sh
chapter_09$ python -m venv chapter11-env2
chapter_09$ source chapter11-env2/bin/activate

(chapter11-env2) chapter_09$ pip install --upgrade pip
(chapter11-env2) chapter_09$ pip install -r requirements.txt
(chapter11-env2) chapter_09$ pip list
```
